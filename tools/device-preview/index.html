Device Preview

Локальная обертка для `localhost`-превью. Удобна для `Expo Web`, обычных React-приложений и любой страницы, которую хочется проверять как телефон в landscape.

Подходит для проверки layout, safe-area отступов, ширины кнопок и поведения экрана на разных мобильных размерах без внешнего сервиса.

`iPhone 15 Pro Max`, `iPhone 15`, `Galaxy S24`, `Pixel 9`, `iPad Mini`, `16:9 Landscape`
Сначала запусти `npm run web`, потом подними этот preview и направь его на адрес Expo, например `http://localhost:8082`.
