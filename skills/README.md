# Skills

Это не системные Codex skills из `~/.codex/skills`, а проектные entrypoints и reusable prompt-файлы.

Главный файл для нового чата:

- [project-onboarding.md](/g:/vs/auto pets/skills/project-onboarding.md:1)

Если нужен полный контекст, агент после него должен читать:

- [AGENTS.md](/g:/vs/auto pets/AGENTS.md:1)
- [docs/onboarding/START-HERE.md](/g:/vs/auto pets/docs/onboarding/START-HERE.md:1)
